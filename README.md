# Gurkenparty
Unsere super tolle website

# Installieren

1. Repository Klonen
 
```sh
git clone https://bricked.com/gurkenparty/website
```

2. Packete installieren

```sh
npm ci
```

3. Server starten

```sh
npm run dev
```
